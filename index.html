YK-Design System

-Yashwanth Karthikeyan

This design system is like a toolbox filled with elements that make digital things look good and work well. It focuses on eight main things: typography (how words look), colors, buttons, cards (boxes with content), alerts (messages), carousel (slideshow), navigation (getting around), and forms. Together, these elements create a smooth and enjoyable journey for users online.
